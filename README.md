The initial prompt for GitHub Copilot to generate optimal sub-agents and agent skills for the repository.

### Usage Guidance
#### Step 1 - Put the [`explore-codebase.ps1`](https://github.com/phanxuanquang/dotnet-project-agent/blob/main/explore-codebase.ps1) file into the root directory of your repository.
#### Step 2 - Paste exactly the content in the [`Initial Prompt.md`](https://raw.githubusercontent.com/phanxuanquang/dotnet-project-agent/refs/heads/main/Initial%20Prompt.md) file into GitHub Coplit and run.
#### Step 3 - Verify the result and enjoy.
